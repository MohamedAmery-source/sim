الموجز (Abstract)

يهدف هذا المشروع إلى تطوير نظام ذكي لتصنيف أنواع النباتات من خلال تحليل صور أوراقها باستخدام تقنيات الرؤية الحاسوبية. يعتمد النظام على دمج تقنيتين مهمتين: مطابقة القالب (Template Matching) لاكتشاف الشكل العام للورقة، وتحليل مصفوفة التكرار الرمادي (GLCM) لاستخلاص خصائص النسيج مثل التباين والتجانس. تم تنفيذ المشروع باستخدام numpy وscikit-image فقط، دون الحاجة إلى OpenCV، مع برمجة بعض الوظائف يدويًا لضمان الشفافية. أظهر النظام دقة جيدة في التمييز بين أنواع مختلفة من النباتات مثل البقدونس، النعناع، والبازيل، مما يجعله أداة فعالة للزراعة الذكية والتعليم البيولوجي.

المقدمة (Introduction)

يعد التعرف على أنواع النباتات من المهارات الأساسية في علم الأحياء والزراعة. ومع التقدم في التكنولوجيا، أصبح من الممكن استخدام الحواسيب لتصنيف النباتات تلقائيًا من خلال تحليل صور أوراقها.

تختلف أوراق النباتات في شكلها ونسيجها وحافتها. يستغل هذا المشروع هذه الاختلافات لبناء نموذج تصنيف ذكي باستخدام تقنيات مثل GLCM لتحليل النسيج وTemplate Matching لاكتشاف الشكل. الهدف هو إنشاء نظام لا يحتاج إلى مكتبات خارجية معقدة، بل يعتمد على برمجة يدوية للفهم الأعمق.

المنهجية (Methodology)

  • البيانات: مجموعة من 30 صورة لأوراق 3 أنواع: البقدونس، النعناع، البازيل.
  • المعالجة: تحويل الصور إلى رمادي، تغيير الحجم إلى 128×128.
  • Template Matching: استخدام scipy.signal.correlate2d لمطابقة الشكل المركزي للورقة.
  • GLCM: حساب التباين، الطاقة، والتجانس باستخدام زوايا متعددة (0°, 45°, 90°, 135°).
  • التصنيف: استخدام خوارزمية KNN بسيطة (k=3) بدون scikit-learn.

الكود (Code) 

import os
import numpy as np
import matplotlib.pyplot as plt
from PIL import Image
from skimage.feature import graycomatrix, graycoprops
from scipy.signal import correlate2d

# تحميل الصور
def load_images(folder_path):
    images = []
    labels = []
    for cls in os.listdir(folder_path):
        class_path = os.path.join(folder_path, cls)
        if not os.path.isdir(class_path):
            continue
        for file in os.listdir(class_path):
            file_path = os.path.join(class_path, file)
            try:
                img = Image.open(file_path).convert('L')
                img = np.array(img.resize((128, 128)))
                images.append(img)
                labels.append(cls)
            except: pass
    return images, labels

# Template Matching
def template_matching_score(image, template_size=(40, 40)):
    h, w = image.shape
    th, tw = template_size
    center_h, center_w = h//2, w//2
    template = image[center_h-th//2:center_h+th//2, center_w-tw//2:center_w+tw//2]
    corr = correlate2d(image, template, mode='valid')
    return np.max(corr)

# GLCM مع زوايا متعددة
def extract_glcm_features(image):
    img = (image / 16).astype(np.uint8)
    glcm = graycomatrix(img, distances=[1], angles=[0, np.pi/4, np.pi/2, 3*np.pi/4], levels=16, symmetric=True, normed=True)
    contrast = np.mean(graycoprops(glcm, 'contrast'))
    energy = np.mean(graycoprops(glcm, 'energy'))
    homogeneity = np.mean(graycoprops(glcm, 'homogeneity'))
    return [contrast, energy, homogeneity]

# تحميل البيانات
X_imgs, y_labels = load_images('images')
features = []
for img, label in zip(X_imgs, y_labels):
    tm = template_matching_score(img)
    glcm = extract_glcm_features(img)
    features.append([tm] + glcm)
X = np.array(features)
y = np.array(y_labels)

المناقشة وتحليل النتائج

  • أظهر Template Matching فعالية في تمييز الشكل المركزي للورقة.
  • كانت أوراق البقدونس أكثر صعوبة في التصنيف بسبب تشابه نسيجها مع النعناع.
